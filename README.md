# My App

Portfolio and booking site for a tattoo artist

Let’s shape your idea into a real app.

Details:
- Who will use this app? The public
- What must it do on day one? User accounts, Add & manage records, Reports / dashboard, Search & filter
- Do users need to sign in? Yes, email + password

Tech stack: React + Express.

## Run

```bash
# frontend
cd frontend && npm install && npm run dev

# backend
cd backend && npm install && npm run dev
```
